## 🚀 部署方式

### 方式一：静态托管（推荐）

将整个 `官网/` 目录上传至任意静态托管平台即可：

- **GitHub Pages** — 推送到仓库，开启 Pages 服务
- **Vercel** — 关联 Git 仓库，自动构建部署
- **Netlify** — 拖拽上传或关联仓库
- **Cloudflare Pages** — 关联仓库，全球CDN加速

### 方式二：传统服务器

将文件上传至 Web 服务器（Nginx / Apache / IIS）的网站根目录即可。

### 方式三：本地预览

```bash
# 使用 Python
cd 官网
python -m http.server 8080

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8080
```

然后访问 `http://localhost:8080`
